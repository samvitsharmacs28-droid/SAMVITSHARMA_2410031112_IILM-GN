#  Energy Analyzer

##  Project Description
**Energy Analyzer** is a Python-based data analysis project that helps monitor, analyze, and visualize the energy consumption of different appliances.  
It calculates total daily energy usage, estimates cost, detects anomalies (high energy usage), and predicts future energy trends using linear regression.  
This project is ideal for understanding data analysis concepts using **Python**, **pandas**, **matplotlib**, and **machine learning (linear regression)**.

---

##  Features
- Analyze appliance-level energy usage  
- Calculate total energy consumption and daily cost  
- Detect anomalies for unusually high energy consumption  
- Visualize energy usage using bar and line charts  
- Apply linear regression for future usage prediction  
- Easy-to-understand terminal and graphical outputs
